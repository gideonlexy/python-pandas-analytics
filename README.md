# Python Pandas Analytics

This repository contains practice solutions for StrataScratch data problems using Python and pandas.

The goal of the repo is to build hands-on data analysis skills by solving interview-style questions with clean pandas workflows such as filtering, grouping, joining, sorting, deduplication, null handling, and aggregation.

## Repo Structure

- `easy/` contains beginner-level StrataScratch-style problems
- `medium/` contains intermediate problems and pandas analysis workflows
- `hard/` contains more advanced data manipulation challenges



## Tools and Concepts

This repo focuses on solving analytics problems with:

- Python
- pandas
- DataFrame filtering
- `groupby`
- `dropna`
- `drop_duplicates`
- sorting and aggregation

## Purpose

This project is intended as a personal practice space for improving:

- pandas fluency
- analytical problem solving
- SQL-to-pandas translation
- data interview preparation

## Running the Code

From the repository root, you can run a solution file with:

```bash
python file_name.py
```

Depending on the platform where the problem came from, some scripts may assume the dataset is already loaded into a pandas DataFrame.

## Notes

- Solutions are organized by difficulty
- The focus is on readable pandas-based analytics solutions
